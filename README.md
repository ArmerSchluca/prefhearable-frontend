# client

Client for PREFEAR.

PREFEAR is a mobile data collection system for capturing individual hearing preferences in real-world environments. The app records user-defined audio preferences together with psychoacoustic parameters, contextual data (e.g., location, environment, time, device information), and optional health-related indicators.

The goal is to build a structured dataset for analyzing and modeling personalized auditory profiles, supporting research in psychoacoustics, context-aware audio perception, and hearing-related well-being.
